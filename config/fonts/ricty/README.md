# How to install Ricty

## Ubuntu

```
% mkdir ~/.fonts
% cp -f ~/dotfiles/config/fonts/ricty/Ricty*.ttf ~/.fonts
% fc-cache -vf
```

## Mac

```
% cp -f ~/dotfiles/config/fonts/ricty/Ricty*.ttf ~/Library/Fonts/
% fc-cache -vf
```
